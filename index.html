CADE Setup

CADE needs a config.js file to start. Create it alongside index.html using the structure below.

// config.js
var CADE_CONFIG = {
  anthropicApiKey: "sk-ant-YOUR-KEY",
  firebase: {
    databaseURL: "https://your-project.firebaseio.com",
    projectId: "your-project"
  },
  model: "claude-haiku-4-5-20251001",
  userPath: "brandon"
};

See config.example.js in the repository for a template. Once your config.js is in place, reload the page.
